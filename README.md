# 2022_DSMP2
Distribution System Mini Project 2

## Start Server 
./Server.py N 

## Client 

### list state 
./client.py g-state 

### change node state 
./client.py g-state <ID> <faulty|nonfaulty>

### give order 
./client.py actual-order <attack|retreat>

### kill node 
./client.py p-kill <ID>

### add node 
./client.py p-add <N>
